Support

Thanks for playing Crazy Discgolf. If you need help, use the options below.

Contact

Best support channel: GitHub Issues

What to include in a bug report

  • Device model and iOS version
  • App version
  • What you expected to happen
  • What actually happened
  • Steps to reproduce the issue

Quick FAQ

  • Q: Do I need an account?
    A: No account is required.
  • Q: Does the app need internet?
    A: Core gameplay works offline. Ads may require network access.
  • Q: Where is my game progress stored?
    A: Progress is stored locally on your device.
